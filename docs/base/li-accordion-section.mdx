# li-accordion-section 元组件

li-accordion-section 组件定义和包含了在 li-accordion 组件中折叠/展开的一段具体内容。其内容由原生 HTML 标签或者 LDS 元组件构成。
组件可通过暴露的 class 属性自定义样式

### 示例

// import { LiAccordionSection } from '@itsi/living-base-react'

### 组件属性

| 属性    | 类型        | 默认值 | 必选 | 描述                                     |
| ------- | ----------- | ------ | ---- | ---------------------------------------- |
| actions | Component[] | 无     | no   | 自定义操作菜单实现，显示在标题的右侧行尾 |
| class | string | 无 | no | 自定义的 CSS 类 |
| label | string | 无 | no | 本节内容的标题 |
| name | string | 无 | no | 本节内容的名称，用于 li-accordiion 组件的 activeSectionName 属性值 |
| title | string | 无 | no | 本节内容的 tooltip 内容 |

### 组件事件

| 事件名称 | 类型 | 描述 |
| -------- | ---- | ---- |
| 无       | 无   | 无   |

### 插槽

| 插槽名称 | 类型        | 默认值 | 必选 | 描述           |
| -------- | ----------- | ------ | ---- | -------------- |
| body     | Component[] | 无     | no   | 折叠的内容部门 |
