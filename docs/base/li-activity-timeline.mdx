# li-activity-timeline 元组件

li-activity-timeline 活动时间轴组件显示即将进行的、当前的和过去的活动

### 示例

### 组件属性

| 属性                | 类型                | 默认值 | 必选 | 描述                                         |
| ------------------- | ------------------- | ------ | ---- | -------------------------------------------- |
| id                  | string              | 无     | yes  | 时间线组件 Id                                |
| itemsExpanded       | string[]            | 无     | no   | 默认需要展开的时间线条目索引组成的数组       |
| isNarrow            | boolean             | 无     | no   | 是否是狭窄模式                               |
| timelineIcon        | Icon                | 无     | no   | 时间线组件每个条目的图标                     |
| timelineActions     | TimelineAction      | 无     | no   | 时间线组件每个条目右侧的操作按钮             |
| timelineMediaFigure | TimelineMediaFigure | 无     | no   | 时间线组件每个条目信息的类型特征标识         |
| timelineMediaBody   | TimelineMediaBody   | 无     | no   | 时间线组件每个条目信息的主题内容             |
| timelineTrigger     | TimelineTrigger     | 无     | no   | 时间线组件每个条目信息的主题内容前缀的复选框 |
| timelineSubtext     | TimelineSubtext     | 无     | no   | 时间线组件每个条目信息的副标题内容           |
| timelineError       | TimelineError       | 无     | no   | 时间线组件错误信息内容位于副标题内容下方     |
| timelineItem        | TimelineItem        | 无     | no   | 时间线组件的每个条目                         |
| descriptionCard     | DescriptionCard     | 无     | no   | 时间线组件详细信息卡片子组件                 |
| fieldInfoListItem   | FieldInfoListItem   | 无     | no   | 时间线组件详细信息卡片中信息条目             |
| narrowContext       | NarrowContext       | 无     | no   | 狭窄风格时间线组件容器                       |
| hasError            | Symbol              | 无     | no   | 是否显示错误信息                             |
