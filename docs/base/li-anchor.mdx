# li-anchor 元组件

li-anchor 是一个[HTML 锚点元素](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/a)的 web 组件实现。
提供了按钮组件相同的视觉外观，包括边框、背景、自定义前后缀图标、突出的色彩显示等

### 示例

import { LiAnchor } from '@itsi/living-base-react'

### 组件属性

| 属性     | 类型                                        | 默认值 | 必选 | 描述                                                                                                               |
| -------- | ------------------------------------------- | ------ | ---- | ------------------------------------------------------------------------------------------------------------------ |
| href     | string                                      | “”     | no   | 超链接指向的 URL 或 URL 片段                                                                                       |
| hrefLang | string                                      | “”     | no   | 用于指定链接文档的人类语言                                                                                         |
| target   | "\_self" / "\_blank" / "\_parent" / "\_top" | “”     | no   | 该属性指定在何处显示链接的资源。 取值为标签（tab），窗口（window），或框架（iframe）等浏览上下文的名称或其他关键词 |
| type     | string                                      | ""     | no   | 该属性指定在一个 MIME type 链接目标的形式的媒体类型                                                                |

### 组件事件

| 事件名称        | 描述           | 类型     |
| --------------- | -------------- | -------- |
| templateChanged | 模板改变时触发 | Function |
| stylesChanged   | 样式改变时触发 | Function |

### 插槽

| 插槽名称 | 描述                       |
| -------- | -------------------------- |
| start    | 在锚点内容之前的提供的内容 |
| end      | 在锚点内容之后提供的内容   |
