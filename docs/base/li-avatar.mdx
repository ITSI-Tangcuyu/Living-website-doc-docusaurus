# li-avatar 元组件

li-avatar 用于图形化地表示用户或对象

### 示例

import Avatar from '../../src/components/Avatar';

<Avatar></Avatar>

### 组件属性

| 属性             | 类型   | 默认值 | 必选   | 描述                                          |
| ---------------- | ------ | ------ | ------ | --------------------------------------------- |
| alternativeText  | string | ""     | no     | 用户鼠标 hover 时描述 li-avatar 组件的文本    |
| fallbackIconName | string | ""     | no     | 当图片加载失败时的默认 ICON                   |
| initials         | string | ""     | no     | 姓名的缩写                                    |
| size             | number | ""     | medium | 大小。有效值为：x-small, small, medium, large |
| src              | string | ""     | no     | 图片 URL                                      |
| variant          | string | ""     | no     | 原型和正方形                                  |
