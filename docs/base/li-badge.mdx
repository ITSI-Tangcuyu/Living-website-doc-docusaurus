# li-badge 元组件

li-badge 徽章组件用于通知用户一个对象的状态或某动作被执行

### 示例

import { LiBadge } from '@itsi/living-base-react'

<LiBadge></LiBadge>

### 组件属性

| 属性      | 类型                                            | 默认值    | 必选 | 描述               |
| --------- | ----------------------------------------------- | --------- | ---- | ------------------ |
| margin    | "auto" \| "none"                                | "auto"    | no   | 控制组件的外边距   |
| theme     | "" \| "default"                                 | ""        | no   | 徽章的主题         |
| variation | "danger" \| "default" \| "success" \| "warning" | "default" | no   | 不同样式风格的徽章 |

### 使用场景

本节为设计人员和开发人员提供了关于在不同上下文中使用此组件的指导原则

import Callout from '../../src/components/Callout'

<Callout emoji="❤️">
  建议使用场景:
  - 为某个动作提供跟新的状态信息
  - 推荐给用户或提醒用户的信息
  - 为新增的对象提供明显的提醒
</Callout>

<Callout type="error" emoji="️🚫">
  避免使用场景：
