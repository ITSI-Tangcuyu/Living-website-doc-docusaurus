# LDS 中的 Design Tokens

Design Token 是一种与技术无关的存储变量的方法。我们使用 Token 而不是硬编码值来确
保不同平台之间更好的 UI 一致性。

### LDS 默认提供的 token 组

| ** 分组 **     | ** 说明 **             |
| -------------- | ---------------------- |
| 颜色           | 主颜色，辅助颜色等     |
| 大小           | 宽度，高度             |
| 间隙和内边距   | 自动布局(间隙和内边距) |
| 边框圆角、阴影 | 所有的边角及阴影       |
| 透明度         | 元素的透明度           |
| 字体类型       | 文本层：采用的字体库   |
| 字体大小       | 文本层：字体大小       |
| 字体粗细       | 文本层：字体的粗细     |
| 文字间距       | 文本层：文字之间的距离 |
| 字体颜色       | 文本层：字体的颜色     |
| z-index        | 元素的覆盖层次         |
| 转换           | 动画场景的转换速度     |
| 媒体查询       | 自适应屏幕大小         |
