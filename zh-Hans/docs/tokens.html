Design Tokens 简介

什么是 Design Tokens？​

Design Token 是设计系统中的视觉设计原子。它们是一组有着统一命名规则的实体，用 于存储视觉设计部分的具体参数，比如 HEX 色值、间距、尺寸的像素等。使用它可以有 帮助为 UI 开发工作维护一套具备可扩展性、一致性的视觉体系。

上面的描述是 Salesforce 在文档中对 Design Token 的定义

通俗来讲：在 WEB 应用的设计领域中，Design Token 是对颜色的色值、间距、尺寸等设计 元素定义的设计变量。对设计变量名称的合理定义可以让设计变量所代表的含义更容易理解 ，也更便于对产品风格的控制。

没有使用 Design Tokens 的 CSS 代码示例：

.button {
  width: 72px;
  height: 48px;
  border-radius: 20rem;
  background-color: #f5f5f5;
  border: 1px solid #cccccc;
}

使用 Design Tokens 的 CSS 代码示例：

.button {
  width: var(--li-size-button-width);
  height: var(--li-size-button-height);
  border-radius: var(--li-border-radius-pill);
  background-color: var(--li-color-500);
  border: var(--li-size-button-tiny-border) solid var(--li-color-border-button);
}

上面 CSS 代码中的背景颜色值从含义模糊的值：#f5f5f5 定义为具备明确含义的设计 变量：--li-color-500, 通过这样的一层抽象定义，可以增强背景颜色值的语义。但这 还远远不够...
