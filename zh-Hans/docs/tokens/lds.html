LDS 中的 Design Tokens

Design Token 是一种与技术无关的存储变量的方法。我们使用 Token 而不是硬编码值来确 保不同平台之间更好的 UI 一致性。

LDS 默认提供的 token 组​

分组 说明
颜色主颜色，辅助颜色等
大小宽度，高度
间隙和内边距自动布局(间隙和内边距)
边框圆角、阴影所有的边角及阴影
透明度元素的透明度
字体类型文本层：采用的字体库
字体大小文本层：字体大小
字体粗细文本层：字体的粗细
文字间距文本层：文字之间的距离
字体颜色文本层：字体的颜色
z-index元素的覆盖层次
转换动画场景的转换速度
媒体查询自适应屏幕大小
